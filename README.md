# CPU-Scheduling-Simulator-using-Python
Mapping the logic of CPU scheduling using Python

In this project, you will design and implement a Python-based CPU Scheduling Simulator that demonstrates how operating systems schedule processes using different scheduling techniques.
This project combines:
1. operating system concepts
2. programming implementation
3. algorithm analysis
4. and responsible use of g

Project Objectives
- Understand the behaviour of CPU scheduling algorithms
- compare the efficiency of scheduling techniques
- analyze process management and resource allocation
- apply operating system concepts in a practical implementation
Student expectations:
- classify scheduling techniques according to their principles
- compate scheduling algorithm performance
- analyze process execution behavior
- and evaluate the strengths and weaknesses of different scheduling methods

Program Requirements
Must use: Python 3.x
You may use:
- Tkinter
- Matplotlib
- Pandas
- Standard Python libraries
You may NOT use:
- prebuilt CPU scheduling simulators
- copied projects from GitHub
- fully AI-generated submissions without modification and understanding
